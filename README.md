# HTML & CSS — Learning Journey

My personal repository for learning the fundamentals of web development, following along with Mosh Hamedani's HTML & CSS course. This repo captures my notes, code examples, and practice files as I work through the material.

---

## Repository Structure

The course is divided into three progressive parts:

### [Part 1 — Web Fundamentals](part-1/)

Foundations of the web and core HTML.

- How the web works (client–server, HTTP, DOM)
- HTML document structure & the `<head>` element
- Text, entities, hyperlinks
- Images, video & audio
- Lists, tables, and containers
- Semantic vs. non-semantic elements

See [part-1/README.md](part-1/README.md) for detailed notes and [part-1/html/](part-1/html/) for code examples.

### [Part 2 — Intermediate](part-2/) _(in progress)_

Building on the fundamentals with more interactive and styled content.

- Forms and form controls
- CSS basics, selectors, and the cascade
- The box model, layout, and positioning
- Flexbox & Grid
- Typography and colors

### Part 3 — Advanced & Project _(upcoming)_

Putting everything together.

- Responsive design & media queries
- Transitions, transforms, and animations
- Best practices & accessibility
- A final end-to-end project

---

## How to Use This Repo

Each part is self-contained with its own folder. To view any lesson:

1. Navigate into the part folder (e.g. [part-1/html/](part-1/html/)).
2. Open any `.html` file directly in a browser, or use the VS Code Live Server extension.
